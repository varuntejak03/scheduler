How to Use AuraPlan

1. Set Planning Horizon

Define your schedule period using the Start Date and End Date fields in the Planning Settings. Select which default weekend days to mark as absent.

2. Download the Template

Click the "Download Template" button. This downloads a structured Excel sheet containing three pre-configured tabs: Settings, Departments, and Resources.

3. Fill in the Data

Open the Excel template and update the worksheets:

  • Departments: Define names, minimum, and maximum work hours required.
  • Resources: Define names, allowed departments, min/max hours, min/max days per department, and individual absent dates (comma-separated, e.g. "2026-07-04, 2026-07-08").

4. Upload & Solve

Click "Upload Data" to import your sheet. If validation passes, click "Generate Schedule". The AI solver will find valid allocations or output a schedule minimizing violations if no solution strictly satisfies all constraints.

5. Review & Export

Cycle through Alternate Solutions to review different valid combinations. Inspect any constraint deviations listed, then click "Export Schedule" to save back to Excel.
